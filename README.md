Deshi Mart Grocery Shop – Full Functional App

Deshi Mart is a full-featured e-commerce mobile app, where users can easily browse products, purchase and track orders. The app is designed with modern UI/UX and user-friendly features.
Authentication & User Management
In this section, the user can manage his account:
Login & Signup: Create a new account and login facility
Profile View & Edit: View your profile and update information
Change Password: Change password for security
Forgot Password: Reset option if you forget your password
Delete Account: Permanently delete your account if necessary
Product Management
To find products easily:
Categories: Sort products into different categories
Products List: View all products at once
Products by Category: View products in a specific category separately
Product Details & Recommendations
Product Details: Detailed information about each product (name, price, description, image, etc.)
Related Products: Show similar or related products as suggestions
User Actions
For user convenience:
Add to Cart: Add favorite products to cart
Add to Favourite: Save favorite products
Order System
To simplify order management:
Create Order: Place an order from the cart
Order Tracking: Current status of the order (processing, shipped, delivered) See
App Appearance
To improve user experience:
Dark Mode: Dark theme for use in low light
Light Mode: Light theme for general use
In short:
Deshi Mart is a modern, user-friendly e-commerce app that makes the user's shopping experience easy, fast and comfortable.
